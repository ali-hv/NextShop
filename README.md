## NextShop
An online-shop API with Django

## How to Run
First, install pipenv:
```
pip install pipenv
```

Clone the repo:
```bash
git clone https://github.com/ali-hv/NextShop.git
cd NextShop
```

Install the requirements
```
pipenv install
```

Run the server
```
cd NextShop
pipenv run python manage.py runserver
```
